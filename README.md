# 📱 SMS Gateway Simulation - DevOps Project

## Project Overview
This project simulates an SMS Gateway system and demonstrates a complete DevOps pipeline using modern tools like Jenkins, Docker, Kubernetes, and AWS.

The application prints a message indicating that an SMS has been successfully sent.

---

## 🎯 Objective
The main goal of this project is to:
- Implement a CI/CD pipeline
- Automate build and deployment
- Containerize the application
- Deploy using Kubernetes

---

## ⚙️ Tech Stack

- **Programming Language**: Java  
- **Version Control**: Git & GitHub  
- **CI/CD Tool**: Jenkins  
- **Containerization**: Docker  
- **Orchestration**: Kubernetes  
- **Cloud (Optional)**: AWS  

---

## 🔁 Project Workflow (CI/CD Pipeline)

1. Developer pushes code to GitHub  
2. Jenkins automatically triggers the pipeline  
3. Docker builds the image  
4. Kubernetes deploys the application  
5. Application runs and prints output
